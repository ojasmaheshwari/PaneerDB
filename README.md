# PaneerDB
Lightweight SQLite like database written in C++

# Build

- `mkdir -p build && cd build`
- `cmake -DCMAKE_EXPORT_COMPILE_COMMANDS=ON -DCMAKE_BUILD_TYPE=Debug -G Ninja ..`
- `ninja`

# Contributing

- Feel free to create issues or MRs
- Please run `clang-format-21` on your code before you create a MR
